HOW to start the 52n-sensorweb-sos-importer
===============================================================================
1 Open any command prompt
2 Go to the folder of the 52n-sensorweb-sos-importer.jar
3 Run "java -jar 52n-sensorweb-sos-importer.jar"
4 It will create a "logs" folder in the same directory where it is running. Make
  sure that you have write permissions in the current folder.
